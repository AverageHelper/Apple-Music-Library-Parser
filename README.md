## Setup

### Prerequesites

This project requires NPM v7 or above. You can check what version you have installed by running `npm -v`:

```sh
$ npm -v
7.20.3
```

### Install Dependencies and Build

```sh
$ npm install
$ npm run build
```

### Run!

```sh
$ npm start /path/to/Library.xml
```
